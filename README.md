Ctrl+W
======

UserScript created for Mush, a browser game.
Mush (French) : http://www.mush.vg
Mush (English) : http://mush.twinoid.com
Mush (Spanish) : http://mush.twinoid.es

Initially created by kill0u, now maintained by badconker.
